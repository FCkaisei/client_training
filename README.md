## ルール
### このゲームの特別ルール
1. 場にある最低値のキューに当てないとコンボが途切れる.
1. 最低値のキューに当たらなくてフォルスになったとしても自球の再配置は行わない
1. 自キューがPocketに落ちてしまったとき,コンボが途切れると同時に自キューの再配置を行う.
